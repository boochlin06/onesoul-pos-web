# OneSoul POS 系統操作流程

本指南將帶您快速了解 OneSoul POS 的核心操作步驟。

## 1. 系統登入與門市選擇
1. 打開 OneSoul POS 網頁（例如透過手機、平板或電腦的瀏覽器）。
2. 在右上角或設定選單確認目前的**選擇門市**（如：竹北、金山），確保讀取與寫入正確的商品庫存與單據。

## 2. 一般結帳流程
1. **商品加入購物車**：
   - 透過搜尋框輸入商品名稱、條碼或關鍵字，即可快速找到商品。
   - 點擊商品加入右側（或下方）的購物車，並可隨時調整購買數量。
2. **選擇會員 (選填)**：
   - 如顧客為會員，可透過手機號碼或姓名搜尋會員資料並綁定此筆交易。
   - 綁定後方可進行「點數折抵」或結帳後「累積點數」。
3. **確認收款方式**：
   - 在購物車下方選擇顧客的付款方式（現金、信用卡、匯款等）。
   - 如會員有足夠點數，可輸入點數折抵金額。的點數將會自動計算扣除。
4. **完成結帳**：
   - 系統自動計算總額後，點擊「結帳」。
   - 成功後系統會清空購物車，並將該筆資料同步至 Google Sheets 後端。

## 3. 當日銷售管理與作廢
1. **查看當日銷售**：
   - 切換至「當日銷售」分頁。
   - 頁面上方會顯示**財務統計**（總營收、現金收取、信用卡、匯款、發放點數等）。
2. **搜尋歷史單據**：
   - 往下滑動即可瀏覽當天所有的交易紀錄，相同交易 ID 的商品會自動群組在一起。
3. **作廢訂單**：
   - 若發生結帳錯誤需退款，可在該筆訂單點擊「作廢」/「刪除」。
   - 系統會自動將該訂單標記，若此單有關聯會員，系統會**自動將發放的點數扣回，或將折抵的點數退還**，確保點數餘額正確無誤。

## 4. 會員與庫存查詢
1. **會員查詢**：切換至會員分頁，可隨時輸入號碼查詢會員當前剩餘點數與基本資料。
2. **庫存/獎項重整**：若後台 Google Sheets 有更新商品資訊，可在 POS 前端點擊「重新整理/同步」按鈕，快速拉取最新的 1000 筆資料庫存。

## 5. 常見問題與異常排除
- **資料未更新**：請確認網路連線正常，並嘗試重整網頁。
- **API 錯誤**：若畫面提示 GAS API 錯誤，請聯繫管理員確認 Google Apps Script 的網址是否正常運作或過期。
