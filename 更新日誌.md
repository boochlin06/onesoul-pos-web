# OneSoul POS 系統更新日誌

## 2026-03-19
### ✨ 新功能 (Features)
- **當日銷售財務統計 (Daily Sales Financials)**：新增每日銷售視圖的財務統計數據，包含總營收、收取現金、信用卡交易額、匯款金額以及點數發放與折抵統計，提供一目瞭然的財務表現。
- **22 項銷售欄位扁平化顯示**：優化銷售紀錄的資料結構，完整呈現 22 項欄位資訊。
- **銷售紀錄快取機制**：加入 sales record caching，提升讀取速度與效能。
- **支援完整會員欄位**：補齊缺失的會員資料欄位，提升會員查詢與顯示的完整性。

## 2026-03-18
### 🚀 重大更新 (Major Update)
- **POS Web 前端現代化重構 (Frontend Migration)**：將原本基於 Google Sheets 的輸入介面，全面遷移至基於 React + Vite 開發的精美網頁應用程式 (SPA)。
- **Tailwind CSS 導入**：使用 Tailwind CSS 建立響應式、現代化且直覺的 UI 介面，優化結帳與購物車體驗。
- **API 與 GAS 後端整合**：前端無縫介接既有的 Google Apps Script (GAS) 後端，持續利用 Google Sheets 作為核心資料庫。
