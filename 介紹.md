# OneSoul POS 系統介紹

OneSoul POS 是一套專為零售門市量身打造的輕量化、高效率銷售點管理系統 (Point of Sale)。本系統獨創運用 **Google Sheets** 作為雲端資料庫（透過 Google Apps Script 提供 API 服務），並搭配現代化的 **React Web 介面**，提供店鋪人員極致流暢的操作體驗。

## 🌟 核心理念
系統旨在解決傳統 POS 系統過於笨重、維護成本高昂的問題。透過無伺服器 (Serverless) 的 Google 雲端生態系，結合極速的網頁前端技術，帶來：
- **零月租費**的資料庫架構（Google Sheets）。
- **即時同步**的雲端試算表，方便管理者隨時查看報表。
- **跨平台**支持，只要有瀏覽器（平板、電腦）即可快速結帳。

## 🚀 系統架構
1. **Frontend (前端 Web 應用)**
   - 採用 **React + Vite** 建置的單頁應用程式 (SPA)。
   - 使用 **Tailwind CSS** 打造現代化且直覺的介面設計。
   - 負責購物車邏輯、點數計算、會員搜尋、當日銷售展示。
2. **Backend (後端 GAS API)**
   - 採用 **Google Apps Script (GAS)** 開發。
   - 負責接收前端請求並讀寫指定的 Google Sheets 工作表。
   - 處理核心業務邏輯：結帳扣庫存、點數更新、作廢訂單同步、資料撈取。

## ✨ 主要功能亮點
- **極致流暢結帳與購物車**：支援快速掃描、搜尋商品並加入購物車，自動計算折扣與應付金額。
- **強大會員與點數系統**：即時查詢會員、發放點數與折抵消費。
- **交易群組化顯示**：各項商品明細會根據交易 ID (checkoutUID) 自動群組為單筆訂單，版面清晰。
- **作廢訂單與點數返還**：在「當日銷售」中一鍵作廢訂單，系統自動比對並退回/扣除會員點數，保證帳務準確。
- **多門市支援**：內建門市切換功能（如：竹北、金山），確保各門市資料獨立與過濾正確。
- **財務數據看板**：當日銷售頁面直接統整今日總營收、現金、刷卡、匯款與點數發放狀況。
